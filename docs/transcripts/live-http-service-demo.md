# Live HTTP-service deployment proof — real Qwen Cloud

Generated by `scripts/live_http_demo.py` against the live Qwen Cloud API.
It teaches the agent over one uvicorn process, **fully restarts the service**,
then recalls over a second process pointed at the same on-disk `MEMORY_DIR`.

- **Endpoint:** https://dashscope-intl.aliyuncs.com/compatible-mode/v1
- **What it proves:** the deployable FastAPI face (`app.py`) keeps cross-session
  memory across a real process restart — recall is backed by the persistent store,
  not by in-process conversation state. Mount `MEMORY_DIR` on a volume and the
  guarantee holds across container restarts (see DEPLOY.md).
- **Reproduce:** `QWEN_API_KEY=sk-... python scripts/live_http_demo.py` (key never committed).
